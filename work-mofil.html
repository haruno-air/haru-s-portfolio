作品詳細

Webサイト / 架空

動物と過ごせるカフェ　Mofil

https://mofil.vercel.app/

ターゲット・目的

ターゲット

・ペットを飼っている人（主に犬）
・ペットと一緒に出掛ける事が多い人
・カフェでゆっくりすることが好きな人

目的

・カフェの存在・雰囲気の周知
・動物連れ込み時の注意点の周知

情報設計・ワイヤーフレーム

いわゆる触れ合いカフェとの違いが分かるよう、コンセプト提示後は注意事項を記載
２行目

デザインカンプ

・動物連れ込みOKなことが分かりやすいように、動物の写真を多用
・実店舗の様子が伝わるように、写真を多用

制作期間

情報設計 １日
ワイヤーフレーム １日
画像作成・デザインカンプ ２日
コーディング ２日

使用ツール

画像編集 Photoshop
ワイヤフレーム・デザインカンプ
コーディング Visual Studio Code
